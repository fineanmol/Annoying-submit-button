# Contribution Rules :books: :

- Do NOT remove other content.
- Try to keep pull requests small to minimize merge conflicts.
- Don't push `Yarn.lock` and `Package-lock.json` in your commit.

        git add .

- Commit your changes

        git push develop new-branch
